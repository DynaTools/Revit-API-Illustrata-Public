# Revit API in Python

La documentazione della **Revit API** decifrata in **Python** — una scheda per classe,
con la corrispondenza C#&nbsp;→&nbsp;Python e un esempio. Gratuito e open source.

!!! tip "Come si usa"
    Cerca una classe nella barra in alto: prova `Document`, `Wall`, `Transaction`,
    `FilteredElementCollector`.

## Cosa trovi in ogni scheda

- **La corrispondenza** — cosa leggi nella documentazione (C#) → cosa scrivi in Python.
- **Il tranello** — ciò che la pagina ufficiale non dice e ti farebbe sbagliare.
- **Lo script** — un esempio pronto da copiare nel RevitPythonShell.

La traduzione C#&nbsp;→&nbsp;Python è generata dalle **7 Regole** del libro.
