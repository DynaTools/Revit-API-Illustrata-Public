Il progetto Revit aperto, in memoria. Nel RevitPythonShell esiste già pronto come doc — il primo argomento di quasi ogni operazione (collector, transazioni, creazione).

La corrispondenza
nella doc (C#)in pythoncosa fa

View ActiveViewdoc.ActiveViewla vista a schermo
bool IsWorkshareddoc.IsWorksharedil modello è condiviso?

Altro: PathName, Create, Delete, FamilyManager…

⚠ Tranello: il doc è il modello in memoria, non il file su disco. PathName è vuoto se mai salvato; modificare richiede Transaction (Legge I).
