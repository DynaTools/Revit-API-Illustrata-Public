# Revit API Illustrata in Python — Codice

Codice ufficiale del libro **Revit API Illustrata in Python** di Paulo Giavoni.

Ogni blocco di codice del libro ha qui il **suo file**, pronto da copiare e
incollare nel **RevitPythonShell** — con l'indentazione esatta (copiare dal
PDF la perde).

Nel libro, sotto ogni blocco, il selo **`</> Codice P.C.n · copia`** porta
direttamente al file corrispondente in [`codice/`](codice/).

- Nome file: `codice-P-C-n.py` — **P** parte, **C** capitolo, **n** numero del
  codice nel capitolo. Esempio: *Codice 6.4.1* → `codice/codice-6-4-1.py`.
- L'elenco completo (con la pagina) è l'**Indice dei codici** in fondo al libro.

Dettagli e note d'uso (engine CPython 3, modalità non-modale per la selezione):
vedi [`codice/README.md`](codice/README.md).
